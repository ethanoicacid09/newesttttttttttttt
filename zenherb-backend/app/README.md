# ZenHerb Backend API

A complete backend for ZenHerb - Natural Wellness & Yoga E-commerce Platform built with FastAPI and PostgreSQL.

## 🚀 Features

- **User Authentication**: JWT-based authentication with registration, login, and profile management
- **Product Management**: Full CRUD operations for herbs and wellness products
- **Course Management**: Yoga courses with video content and enrollment system
- **Shopping Cart**: Complete cart functionality with quantity management
- **Order Management**: Order creation, tracking, and history
- **Wishlist**: Save favorite products
- **Contact Form**: Customer inquiry management
- **Admin Dashboard**: Statistics and management features

## 🛠️ Tech Stack

- **Backend**: FastAPI (Python)
- **Database**: PostgreSQL
- **ORM**: SQLAlchemy
- **Authentication**: JWT (python-jose)
- **Password Hashing**: Bcrypt (passlib)
- **Validation**: Pydantic
